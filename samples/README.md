# Sample Test Fixtures

This folder contains sample trial balance inputs and expected report outputs for manual testing.

## Files

- `trial-balance/TB-Q1-2025-26.xlsx`
  - Current quarter trial balance covering common Schedule III categories.
  - Includes mapped ledgers such as Equity Share Capital, Flat Sales, Salary Expense, Fixed Assets, Bank Account, Trade Payables, Borrowings, etc.
  - Includes one deliberately unmapped ledger: `Secret Suspense Account`.

- `trial-balance/TB-Q4-2024-25.xlsx`
  - Prior quarter trial balance for comparative testing.

- `trial-balance/TB-Annual-2024-25.xlsx`
  - Annual prior period trial balance for year-ended comparative testing.

- `expected-report/ideal-current-quarter-report.xlsx`
  - Expected workbook generated from `TB-Q1-2025-26.xlsx` without comparatives.

- `expected-report/ideal-current-quarter-report-with-comparatives.xlsx`
  - Expected workbook generated from `TB-Q1-2025-26.xlsx` using `TB-Q4-2024-25.xlsx` and `TB-Annual-2024-25.xlsx` as comparative periods.

## How to use

1. Upload `samples/trial-balance/TB-Q1-2025-26.xlsx` to the app for the current period.
2. Upload the comparative files and ensure they are associated with prior periods.
3. Generate the report in the app.
4. Download the generated workbook and compare it against the expected workbook in `samples/expected-report/`.

## Notes

- The expected report files were generated by the system in its current state and serve as a baseline for verifying behavior.
- Use the comparative report file when testing the new Sprint 3 comparative period workflow.
